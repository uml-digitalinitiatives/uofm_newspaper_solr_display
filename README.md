SUMMARY
-------

UofM Newspaper Solr Display

Mimicks the default Islandora display for islandora:newspaperCModel objects,
but uses Solr queries instead of Sparql.

REQUIREMENTS
------------

Dependent on:

* islandora\_newspaper
* islandora\_solr

INSTALLATION
------------

Install as any other Drupal module.
